# 🌟 Why OpenSource Mentor?

Open source powers the modern software ecosystem, yet contributing to open-source projects remains one of the biggest challenges for new developers.

Understanding a large repository, learning its architecture, finding the right issue, and preparing a first contribution often takes **hours or even days**.

**OpenSource Mentor** reduces this onboarding journey to **minutes** by combining AI Agents, Workflows, and Persistent Knowledge using the **Lemma Platform**.

---

# 🎯 Problem Statement

Developers struggle to contribute to open-source because they need to:

- 📚 Read thousands of lines of documentation
- 🏗 Understand unfamiliar architectures
- 🔍 Find beginner-friendly issues
- 🤔 Learn project-specific concepts
- 📝 Write quality pull requests

OpenSource Mentor solves this through an AI-powered developer workspace.

---

# 💡 Solution

Import any public GitHub repository and the platform automatically:

✅ Understands the repository

✅ Generates architecture explanations

✅ Creates a searchable knowledge base

✅ Finds beginner-friendly issues

✅ Builds personalized learning paths

✅ Reviews pull requests

✅ Tracks contribution progress

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/shaiks2004/Open-Source_Mentor.git
cd Open-Source_Mentor
```

Import the Pod

```bash
lemma pod import .
```

Deploy

```bash
lemma apps deploy frontend apps/frontend/source
```
